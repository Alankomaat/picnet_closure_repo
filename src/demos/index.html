PicNet Closure Repository - Demo

This is a demonstration of all the controls / compoenents in the picnet-closure-repo project. Url http://code.google.com/p/picnet-closure-repo/ .

  • pn.ui.DateRangePicker: A simple date range selector, allows unbounded ranges.
  • pn.ui.TabSlideMenu: Hide menus, panes, etc off the side of the screen and display when the user clicks or hovers the appropriate tab.
  • pn.ui.filter.TableFilter: A real time column filtering on a regular Html table.
