pn.seq.Seq

The pn.seq.Seq class is an implementation of the .Net LINQ framework in JavaScript and its implementation was heavily influenced by Jon Skeets excellent study of the .Net LINQ framework. Note: this implementation does not use deferred execution so it is not as performant as it could be.

LINQ Explorer

There is a sequence in memory loaded. This sequence is as follows:
var seq = new pn.seq.Seq([
... ]);
